

THE BOX MODEL IS ONE OF THE MOST FUNDAMENTAL PART OF CSS 
THAT'S BECAUSE IT'S THE MAIN FACTOR THAT DEFINES HOW ELEMENTS ARE DISPLAYED ON A WEB PAGE AND ALSO HOW THEY ARE SIZED 

---> SO ACORDING TO THE BOX MODEL:

- EACH AND EVERY ELEMENT ON A WEBPAGE CAN BE SEEN AS A RECTANGULAR BOX AND EACH BOX HAVE A WIDTH, A HEIGHT, PADDING, MARGINS AND A BORDER 

OBS: 

WIDTH - A PROPRIEDADE WIDTH DETERMINA A LARGURA DA AREA D CONTEUDO DE UM ELEMENTO. 
A ÁREA DE CONTEUDO FICA DENTRO DO PREENCHIMENTO, DA BORDA, DA MARGEM E DE UM ELEMENTO 

HEIGHT - DETERMINA A ALTURA DA ÁREA DE CONTEÚDO DE UM ELEMENTO 


PADDING - A PROPRIEDADE PADDING É RESPONSAVEL POR DEFINIR UMA DISTANCIA INTERNA ENTEE UM ELEMENTO E SUA BORDA.
CRIANDO ESSA LACUNA ENTRE O ELEMENTO E SEU CONTEÚDO, AUMENTARÁ O TAMANHO DO ELEMENTO OU DIMINUIRÁ SEU CONTEÚDO.
MAS, POR PADRAO, O TAMANHO DO ELEMENTO AUMENTA DE MODO QUE SE ACOMODE A ESSA DISTANCIA.

MARGIN - A PROPRIEDADE QUE DEFINE A ÁREA DE MARGEM NOS QUATRO LADOS 
DO ELEMENTO. É UMA ABREVIAÇÃO QUE DEFINE TODAS AS MARGENS INDIVIDUAIS DE UMA VEZ SÓ: MAGIN-TOP, MARGIN-RIGHT, MARGIN-BOTTOM E MARGIN-LEFT

BORDER - É UMA PROPRIEDADE QUE ADICIONA UM CONTORNO EM VOLTA DE UM ELEMENTO HTML.
ELA PODE SER APLICADA EM TODOS OS LADOS DE UM ELEMENTO DA PAGINA E TER CARACTERISTICAS DIFERENTES, COMO TAMANHO, CORES E ESTILOS.


WE CAN SPECIFY ALL OF THESE USING CSS PROPERTIES 
BUT NOTE THAT THEY ARE ALL OPTIONAL!!!

- SO THERE CAN BE BOXES WITH NO MARGINS OR NO PADDINGS OR NO BORDER AT ALL 

---=============================================================================================================================================

- CONTENT: TEXT, IMAGES ETC. 
- BORDER: GOES AROUND THE PADDING AND THE CONTENT
- MARGIN: SPACE BETWEEN BOXES -----> EX: IF WE WANT TO CREATE SOME SPACE BETWEEN TWO ELEMENTS 
WE WILL SIMPLY USE THEIR MARGIN PROPERTIES IN CSS 

- FILL AREA: AREA THAT GETS FILLED WITH BACKGROUND COOR OR BACKGROUND IMAGE 

THE TEXT CONTENT AND IMAGES GO INSIDE THE CONTENT BOX
THE SAME DOES NOT APPLY TO BACKGROUND IMAGES OR THE BACKGROUND COLOR OF A BOX 

THESE PROPERTIES WILL ACTUALLY BE APPLIED NOT ONLY TO THE CONTENT BOX, 
BUT TO THE ENTIRE FILL AREA (WHICH ALSO INCLUDES THE PADDING AND THE BORDER---- BUT NOT THE MARGIN!!!)


BY DEFAULT, SOME ELEMENTS GET SOME MARGIN AUTOMATICALLY IF WE DON'T DEFINE THEM - THAT CAN CAUSE PROBLEMS 


IMPORTANT::: RESET ALL OF THE MARGINS AND ALL OF THE PADDINGS ON ALL ELEMENTS BEFORE DOING ANYTHING ELSE 
THIS IS CALLED A GLOBAL RESET 

AND FOR THAT WE NEED TO SELEC ALL OF THE ELEMENTS ANF THE SELECTOR IS *
IF THE * WE CAN SELECT ALL THE ELEMENTS ON THE PAGE 

*{
    margin: 0;
    padding: 0; 
    box-sizing: border-box; 
}
